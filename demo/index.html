Features

Built for modern payments

Everything you need to move money across chains without friction.

Gasless via Gas Station

Zero gas, all sponsored. Circle Gas Station covers every transaction. No native tokens needed.

Non-Custodial (Passkey)

TouchID/FaceID, no seed phrases. Keys in device secure enclave. Full self-custody.

Modular AA (MSCA)

ERC-4337 + ERC-6900. Plug any logic module. Atomic batching of approve + swap + bridge.

Batch Payouts

50 recipients, 5 chains, one API call. Volume discount at 0.25%.

Any-Token via LI.FI

Deposit ETH, receive DAI, settle in USDC. LI.FI aggregates DEXes for optimal swaps.
